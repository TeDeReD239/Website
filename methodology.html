Задание №1 — Освоение функции выдавливания

🎯 Цель: Освоение функции выдавливания в КОМПАС-3D.
🛠️ Инструменты: Элемент выдавливания, эскиз, прямоугольник, окружность, авторазмер, геометрические ограничения, плоскости ZX, верхняя и боковая плоскости.
📌 Краткое описание: Создание детали, состоящей из прямоугольного основания, цилиндрического выступа и призматического элемента, расположенного на боковой грани.
💬 Комментарий: В задании отрабатываются базовые навыки построения эскизов на разных плоскостях, использование операции выдавливания с разной глубиной, включая выдавливание «до ближайшей поверхности».

Задание №2 — Освоение функции вырезания выдавливанием

🎯 Цель: Освоение операции вырезания выдавливанием.
🛠️ Инструменты: Элемент выдавливания, вырезание выдавливанием, эскиз, прямоугольник, окружность, авторазмер, плоскости.
📌 Краткое описание: Создание детали на основе прямоугольного основания с добавлением цилиндрического прилива, сквозного отверстия и бокового паза, созданного вырезанием до ближайшей плоскости.
💬 Комментарий: Задание закрепляет навыки выдавливания и знакомит с обратной операцией — вырезанием. Особое внимание уделяется созданию сквозных отверстий и пазов.

Задание №3 — Скругления, фаски и массивы

🎯 Цель: Освоение операций скругления, создания фасок и построения массивов.
🛠️ Инструменты: Скругление, фаска, массив по сетке, вырезание выдавливанием, эскиз, окружность, прямоугольник.
📌 Краткое описание: Создание детали с двумя отверстиями (сквозным Ø10 мм и углублением Ø20 мм глубиной 5 мм), фасками и скруглениями на отверстиях, а также массивом из четырёх вертикальных выступов по углам основания.
💬 Комментарий: Задание демонстрирует возможности массового создания элементов через массив 2×2, а также применение фасок и скруглений для обработки кромок.

Задание №4 — Операция вращения

🎯 Цель: Освоение операции вращения и работы с дугами в эскизе.
🛠️ Инструменты: Элемент вращения, эскиз, прямоугольник, дуга по трём точкам, усечь кривую, фаска, плоскость ZY.
📌 Краткое описание: Создание тела вращения на основе эскиза, включающего прямоугольник 15×5 мм, смещённый на 5 мм от оси, и дугу радиусом 12 мм. На внутреннем отверстии выполнена фаска 2 мм.
💬 Комментарий: Задание знакомит с альтернативным способом создания деталей — вращением эскиза вокруг оси. Особое внимание уделено построению дуги и использованию инструмента «Усечь кривую».

Задание №5 — Сборка из моделей предыдущих заданий

🎯 Цель: Освоение создания сборки из нескольких деталей.
🛠️ Инструменты: Сборка (ассемблер), добавление компонентов, сопряжения (соосность, совпадение), фиксация положения.
📌 Краткое описание: Сборка изделия из четырёх деталей, созданных в предыдущих заданиях. Детали последовательно фиксируются относительно центрального цилиндра Детали №1 с помощью сопряжений «Соосность» и «Совпадение».
💬 Комментарий: Итоговое задание, объединяющее все ранее созданные детали в единую сборку. Отрабатываются навыки работы с сопряжениями и проверки пересечений компонентов.

Задание №6 — Выдавливание по сечениям (лофт) и оболочка

🎯 Цель: Освоение операции выдавливания по сечениям (лофт) и инструмента «Оболочка».
🛠️ Инструменты: Выдавливание по сечениям, вспомогательные плоскости, правильный шестиугольник, поворот, оболочка, скругление.
📌 Краткое описание: Создание вазы методом выдавливания по сечениям на трёх плоскостях (0, 30, 60 мм) с шестиугольниками разного размера (средний повёрнут на 45°). Операция «Оболочка» с толщиной 2 мм создаёт полость.
💬 Комментарий: Задание знакомит с мощным инструментом «Выдавливание по сечениям», позволяющим создавать плавные переходы между разными формами. Операция «Оболочка» превращает модель в полое изделие за один клик.
